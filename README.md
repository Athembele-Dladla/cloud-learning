# ☁️ Cloud Projects

A collection of beginner cloud projects built while learning cloud platforms like AWS, Google Cloud, and Firebase.

## 📁 Projects

| Project | Description | Platform | Status |
|---|---|---|---|
| `static-website` | HTML/CSS site hosted on cloud storage | AWS S3 / Firebase | 🚧 Currently in progress |
| `serverless-hello-world` | HTTP-triggered serverless function | AWS Lambda / GCF | 🔜 Coming Soon |
| `url-shortener` | Short URL generator using a cloud database | AWS + DynamoDB | 🔜 Coming Soon |
| `weather-dashboard` | Fetches and stores weather data | GCP + Firestore | 🔜 Coming Soon |
| `file-upload-app` | Upload files to cloud storage | Firebase Storage | 🔜 Coming Soon |

## 🛠️ Tools & Platforms
- **AWS** — Lambda, S3, DynamoDB
- **Google Cloud** — Cloud Functions, Firestore
- **Firebase** — Hosting, Storage, Auth
- **Languages** — Python, JavaScript

## 🎯 Goal
Work through beginner cloud concepts including:
- Cloud storage & static hosting
- Serverless / event-driven functions
- Cloud databases (NoSQL & SQL)
- REST APIs on the cloud
- CI/CD pipelines

🚧 Currently in progress — projects will be added as they are completed.
